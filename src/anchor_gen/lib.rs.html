9//! # Usage
10//!
11//! First, add the following to a `Cargo.toml` file in a new crate:
12//!
13//! ```toml
14//! [dependencies]
15//! anchor-gen = "0.31.1"
16//! ```
17//!
18//! Then, in `lib.rs`, write:
19//!
20//! ```skip
21//! anchor_gen::generate_cpi_crate!("../../examples/govern-cpi/idl.json");
22//! ```
23//!
24//! This will generate a fully functional Rust CPI client for your IDL.
25//!
26//! Usage examples can be found in the [examples/](https://github.com/saber-hq/anchor-gen/tree/master/examples) directory.
27//!
28//! Note: This does not work on legacy IDLs. To migrate a legacy IDL, use `anchor idl convert idl.json`.
