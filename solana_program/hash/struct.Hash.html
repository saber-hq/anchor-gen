Struct Hash

Source 
pub struct Hash(/* private fields */);
Expand description

A hash; the 32-byte output of a hashing algorithm.

This struct is used most often in solana-sdk and related crates to contain a SHA-256 hash, but may instead contain a blake3 hash.
